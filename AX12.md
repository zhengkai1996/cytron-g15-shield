# AX12() #

## Description ##
This is the constructor for the AX12 class, which inherits from G15 class.

## Include ##
G15.h

## Prototype ##
AX12(byte ID);

## Parameters ##
**ID**: AX12's ID

## Returns ##
None

## Example ##
AX12 servo2(2);
//This is declaration of an object named servo2 which belongs to AX12 class. It has the ID of 2.
